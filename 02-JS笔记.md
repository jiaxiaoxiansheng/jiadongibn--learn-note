####排序算法
+ 冒泡排序:两两比较判断是否交换，两两变量交换时，要申明一个变量来接收其中一个的值

+ 比较排序：假设当前数组中的第一个值是最小值，然后拿这个最小值和后面的项逐一比较，如果后面的某一个值比假设的值还小，说明假设错了，我们把假设的值进行替换。最后得到的结果就是我们想要的。
</br>
+ 递归思想：递归一定要有跳出条件，要不然会一直循环
   + 5!= 4!*5
   + 4!= 3!*4
   + 3!= 2!*3
   + 2!= 1!*2   5! = 4!*5=(3!*4)!*5
   + n! = (n-1)!*n
   ```
       function jc (num){
         // 递归的跳出条件
           if(num===1){
           return 1
           }
           return jc(num-1)*num
         }
     
    ```
